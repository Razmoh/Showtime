## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Versions

nest - 8.2.5 \
npm - 8.6.0

## Installation

```bash
$ npm install
```

## Dependances

```bash
$ npm install --save @nestjs/mongoose mongoose
$ npm install --save class-validator class-transformer
```

## Running the app

```bash
# development
$ npm run start
# HotReload
$ npm run start:dev

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

App use a local MongoDB: \
-> Need to **Install** and **Run** <a href="https://www.mongodb.com/docs/manual/administration/install-community/" target="_blank">MongoDB</a>

## MongoDB (MAC)

Installation

```bash
# development
$ xcode-select --install
# HotReload
$ brew tap mongodb/brew

# watch mode
$ brew update

# production mode
$ brew install mongodb-community@5.0
```

Run

```bash
#Start
$ brew services start mongodb-community@5.0

# Stop
$ brew services stop mongodb-community@5.0

# Check
$ brew services list
```

All info <a href="https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/#run-mongodb-community-edition" target="_blank">Here</a>

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
